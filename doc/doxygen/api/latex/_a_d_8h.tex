\subsection{Detailed Description}
Multiplexes the A\-D/\-C to read analog ports. \begin{DoxyAuthor}{Author}
Max Dunne 

David Goodman
\end{DoxyAuthor}
Software module to enable the Analog to Digital converter of the Uno32 boards. All analog pins are are Port V and Port W, with an additional analog input for the battery voltage (through a 10\-:1 divider).

\begin{DoxyNote}{Note}
Analog pins automatically take over digital I/\-O regardless of which T\-R\-I\-S state it is in. There remains an error in the A\-D\-C code such that if all 12 pins are enabled, one of them does not respond.

A\-D\-\_\-\-T\-E\-S\-T (in the .c file) conditionally compiles the test harness for the code.

Make sure it is commented out for module useage.
\end{DoxyNote}
