\subsection{Detailed Description}
Multiplexes a P\-W\-M line into many channels. \begin{DoxyAuthor}{Author}
Max Dunne 

David Goodman
\end{DoxyAuthor}
Software module to run the P\-W\-M module of the Uno32. The available pins for which the P\-W\-M works are \#defined below (Port\-Z-\/6, Port\-Y-\/4,10,12, and Port\-X-\/11), and are set by the hardware (cannot be modified).

\begin{DoxyNote}{Note}
Module uses T\-I\-M\-E\-R2 for its interrupts.

P\-W\-M\-\_\-\-T\-E\-S\-T (in the .c file) conditionally compiles the test harness for the code. Make sure it is commented out for module useage.
\end{DoxyNote}
