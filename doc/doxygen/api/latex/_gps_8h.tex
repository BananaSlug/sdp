\subsection{Detailed Description}
Sensor interface for the G\-P\-S. \begin{DoxyAuthor}{Author}
David Goodman
\end{DoxyAuthor}
Module that wraps the G\-P\-S in a state machine, which ocassionally pulls location and velocity data from the G\-P\-S over U\-A\-R\-T. Also, this module can be configured to communicate with a command station to reduce error and improve accuracy.
