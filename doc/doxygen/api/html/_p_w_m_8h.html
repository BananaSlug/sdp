Detailed Description

Multiplexes a PWM line into many channels.

Author
Max Dunne
David Goodman

Software module to run the PWM module of the Uno32. The available pins for which the PWM works are #defined below (PortZ-6, PortY-4,10,12, and PortX-11), and are set by the hardware (cannot be modified).

Note
Module uses TIMER2 for its interrupts.
PWM_TEST (in the .c file) conditionally compiles the test harness for the code. Make sure it is commented out for module useage.
