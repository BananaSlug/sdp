Detailed Description

Multiplexes the AD/C to read analog ports.

Author
Max Dunne
David Goodman

Software module to enable the Analog to Digital converter of the Uno32 boards. All analog pins are are Port V and Port W, with an additional analog input for the battery voltage (through a 10:1 divider).

Note
Analog pins automatically take over digital I/O regardless of which TRIS state it is in. There remains an error in the ADC code such that if all 12 pins are enabled, one of them does not respond.
AD_TEST (in the .c file) conditionally compiles the test harness for the code.
Make sure it is commented out for module useage.
